# AI Quiz Generator System

An AI-powered web application that automatically generates quizzes based on a given topic.
This project uses **FastAPI** for the backend and **Groq LLM** to generate quiz questions dynamically.

## 🚀 Features

* Generate quiz questions using AI
* FastAPI-based backend
* Simple web interface (HTML + CSS)
* Topic-based quiz generation
* Lightweight and fast response

## 🛠️ Tech Stack

* **Python**
* **FastAPI**
* **Groq API (LLM)**
* **HTML**
* **CSS**
* **Uvicorn**
